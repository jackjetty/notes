# 重要（基本）
* 查找
    * Ctrl+F: 查找 （Find）
    * Ctrl+Shift+F: 在文件中查找
    * F3: 查找下一个
* Ctrl+K,C: 注释选定内容 （Comment）
* Ctrl+K,U: 取消选定内容注释 （UnComment）
* Ctrl+J：智能提示
* Ctrl+K+F/D：格式化代码
* 调试
    * F5: 启动调试
    * F10: 逐过程 
    * F11: 逐语句
* 两下Tab（C++中貌似是输完关键字后再一下就行了）：快速代码段（if，switch，for，foreach循环，try，绑定事件方法）
* 列编辑
    * Shift+Alt+Arrow Key
    * Shift+Alt+鼠标选择
