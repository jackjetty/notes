* 安装Python
    * 传统的还是用2，2和3差别很大
    * 官网下载exe
* 安装pip包管理器
    * python -m pip install -U pip
    * 将C:\Python27\Scripts加到环境变量中
* IDE
    * 常用PyCharm，社区免费开源版即可
        * 按文件夹打开Project
        * 项目根目录会有.idea目录，包含项目配置
        * 通常要在Settings中进行项目配置
            * Project Interpreter
                * 下面的工具程序都会被放到项目的/venv/Scripts/xenv目录下
                * python.exe
                * pip
                * setuptools
                * wheel
